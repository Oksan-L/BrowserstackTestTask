## **Test ID**
TC-002

## **Title**
Login empty text

## **Description**
Verify that the login form shows a warning message if the username or password field is left empty.

## **Preconditions**
- Android emulator or BrowserStack device is running  
- Appium server is running and connected  
- Android Native Demo App is installed on the device  

## **Test Steps**
1. Navigate to the **Login** screen  
2. Ensure the **username** field is empty  
3. Ensure the **password** field is empty  
4. Tap the **"Login"** button  

## **Expected Results**
1. Login screen is displayed  
2. Username and password fields are empty  
3. Warning messages appear below each empty field indicating that input is required

- **Test:** `FormsScreen.test.js`
