## **Test ID**
TC-001

## **Title**
Form interactions in Android Demo App

## **Description**
This test verifies that a user can interact with form elements in the Android Native Demo App, including clicking buttons, entering text, toggling switches, selecting from dropdowns, and confirming alert dialogs.

## **Preconditions**
- Android emulator or BrowserStack device is running  
- Appium server is running and connected  
- Android Native Demo App is installed on the device  

## **Test Steps**
1. Tap the **"Forms"** button  
2. Tap the text input field  
3. Clear the text input field  
4. Enter **random generated text** into the text input field  
5. Tap the **switch** element to toggle it  
6. Tap the **Dropdown** element  
7. Select **"Appium is awesome"** from the dropdown  
8. Tap the **"button-Active"** button  
9. Tap **"OK"** on the alert dialog  

## **Expected Results**
1. Forms page is opened and displayed  
2. Text input field is focused  
3. Text input field is empty  
4. Text input field contains the newly entered random text  
5. Switch state changes successfully  
6. Dropdown menu opens  
7. Option **"Appium is awesome"** is selected  
8. Alert dialog appears  
9. Alert dialog closes and user is returned to the Forms page
